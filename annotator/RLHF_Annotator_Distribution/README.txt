RLHF Annotator 
================ 
 
This executable is a standalone application for annotating conversation logs for RLHF training. 
 
Instructions: 
1. Double-click RLHF_Annotator.exe to launch the application 
2. Use "Open Log File" to select a conversation log to annotate 
3. Navigate through conversations and provide feedback 
4. Use "Export Data" to save your annotations
